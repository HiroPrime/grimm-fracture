# Outline & Script Prompt Pack

Paste into the Grimm Fracture Writer Gem after knowledge files are loaded.

---

## `/ideas` — Idea sprint

```
/ideas
Context: Leap 1 Overgrown Wonderland.
Goal: [e.g. mid-arc set piece / page 1 hook / Alice-Red conflict]
Generate 8 options. For each: 1-line pitch, why it fits canon, risk, ship-speed score 1–5.
Rank top 3. Recommend 1. End with KEEP / TWIST / SCRAP buttons.
Label all as DRAFT.
```

---

## `/outline` — Leap outline

```
/outline
Build a 24–32 page Leap 1 outline using LEAP_01_1888.md.
Structure by page clusters (not chapters).
Each cluster: goal, obstacle, character spotlight, visual set piece, cliff.
Respect series physics. No new permanent costume redesigns.
Mark speculative world details DRAFT.
```

---

## `/script N` — Single page script

```
/script [N]
Write web comic page [N] panel-by-panel.
Format:
PAGE N — TITLE
Panel 1: [shot] / [action] / [dialogue/caption/SFX]
...
Max 7 panels. End on a micro-cliff.
Use RED_1888 and ALICE_1888 voice locks.
After script, give a 4-bullet art brief for the page.
```

---

## Batch pages

```
Script pages [A] through [B] for Leap 1.
Keep continuity of injuries, weapons, and location.
After each page: one-line purpose + cliff.
Then list any continuity risks across the batch.
```

---

## Dialogue polish

```
Rewrite this page's dialogue sharper and shorter.
Alice = manic precise. Red = blunt exhausted.
Keep meaning. Cut 20% words minimum.
```

---

## `/critique` — Story doctor

```
/critique
Review this beat sheet / script for:
1) Hook strength
2) Character consistency
3) Visual staging clarity
4) Web serialization cliffs
5) Canon violations
Be blunt. Offer a fixed version of the weakest page.
```
