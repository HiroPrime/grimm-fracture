# Workflow: Iteration Loops

Goal: fewer than **3 passes** from draft → shippable.

---

## Loop A — Story (Writer Gem)

```
Pass 1: Generate (outline or script)
Pass 2: Critique (/critique) + your KEEP/TWIST/SCRAP
Pass 3: Final polish only (dialogue cut / cliff sharpen)
STOP. Ship to art.
```

If still weak after Pass 3 → **SCRAP structure**, don’t micro-edit forever.

---

## Loop B — Art (Art Director Gem)

```
Pass 1: Generate with LOCKED VISUAL BLOCK + reference image
Pass 2: Consistency repair prompt (fix drifts only)
Pass 3: Crop/lighting/panel framing tweak OR accept
STOP. Do not redesign costume mid-page.
```

**Hard stop rule:** if Pass 2 still breaks identity anchors → regenerate from Pass 1 with stricter negative prompt, don’t keep “fixing” a wrong face.

---

## Feedback language cheat sheet

Use short, mechanical notes Gemini can obey:

| You say | Meaning |
|---------|---------|
| `TWIST: more vertical` | Rooftops / height / falling |
| `TWIST: cut 30% dialogue` | Visual storytelling |
| `TWIST: Red lead` | Alice support this page |
| `FIX LOCK: keyhole brands` | Identity repair |
| `FIX LOCK: axe tape handle` | Prop repair |
| `ENERGY: +panic` | Emotion dial |
| `ENERGY: -quip` | Tone dial |
| `SHIP IT` | End loop |

---

## Skill ampers (use Gemini as coach)

Weekly 15-min drills:

1. **Staging drill:** one silent 5-panel page (no dialogue)  
2. **Voice drill:** rewrite same argument as Red-only then Alice-only  
3. **Thumbnails drill:** describe 6 thumbnail compositions before scripting  
4. **Cliff drill:** write 5 alternate last panels for the same page  
5. **Consistency drill:** list 5 drifts in a bad generation, then repair prompt  

Ask Gemini after each drill:  
`What craft skill did I just train, and what’s tomorrow’s 10-min drill?`
