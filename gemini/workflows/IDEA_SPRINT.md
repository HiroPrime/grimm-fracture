# Workflow: Idea Sprint (10–20 min)

Use when you need options fast — hooks, set pieces, Leap pitches, or stuck mid-arc.

## Setup

Gem: **Grimm Fracture Writer**  
Mode: `/ideas`

## Sprint card (copy)

```
IDEA SPRINT
Constraint: stay inside SERIES_BIBLE + Leap 1 locks
Axis to explore: [hook / enemy set piece / Red-Alice conflict / Keystone twist / humor-dark beat]
Quantity: 8
Timebox: one reply, then I feedback
```

## Feedback protocol (immediate iteration)

Reply with only one of:

- `KEEP #3` — develop into outline  
- `TWIST #3` + note (`make it more vertical` / `less dialogue` / `raise stakes`)  
- `SCRAP` — new 8, avoid X  
- `HYBRID 2+5` — merge  

Gemini must regenerate in ≤1 reply using that instruction.

## Play modes (rotate)

1. **Opposite day** — invert the obvious beat; keep canon physics  
2. **Prop-first** — story grows from axe / flask / razor-card  
3. **Location-first** — tonic gardens / spade alley / rail yard  
4. **Cost-first** — what does winning this page cost Red or Alice?  
5. **Enemy POV** — one page from Stalker or Ironclad (DRAFT only)

## Promotion rule

Nothing from a sprint is canon until you say:

`LOCK: put this into leaps/LEAP_01_1888.md under [section]`

Then ask Gemini for `/lock` paste-ready markdown.
