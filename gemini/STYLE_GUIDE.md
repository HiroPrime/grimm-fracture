# Grimm Fracture — Style Guide (CANON)

## Master art style lock (paste into every image prompt)

```
Frenetic comic illustration style: scratchy ink outlines, heavy cross-hatching,
gritty Victorian fairy-tale noir, aged parchment texture, muted earthy palette
(oxblood red, dirty teal, soot brown, parchment tan), dramatic exhausted faces,
high contrast shadows, concept-art model-sheet clarity, NOT clean anime, NOT
soft Disney, NOT photoreal CGI, NOT cute chibi.
```

## Style pillars

| Pillar | Do | Don’t |
|--------|----|-------|
| Line | Scratchy ink, energetic edges | Clean vector outlines |
| Shade | Cross-hatching, grit | Soft airbrush gradients only |
| Color | Muted + toxic accent (Alice chemicals) | Pastel candy Wonderland |
| Paper | Aged parchment / stained page feel | Pure white comic stock |
| Faces | Exhaustion, mania, defiance | Glossy model beauty |
| Action | Heavy, consequential, dirty | Floaty wire-fu elegance |

## Series palette tokens

- **Blood / Red identity:** `#8b0000`, oxblood, scuffed leather red  
- **Alice botanic:** `#2e4a22`, dirty teal / petrol blue, toxic green vial glow  
- **Syndicate:** white-iron, black spade, rust brown, steam orange sparks  
- **Atmosphere:** smog grey, parchment tan, soot  

## Panel language (web comic)

- **Splash:** arrival, Keystone reveal, major death/threat  
- **Wide establishing:** Overgrown Wonderland streets (Victorian + mutated flora)  
- **Close-up currency:** eyes (Alice manic glare / Red exhausted glare)  
- **Prop hero shots:** axe, flask, stun-baton, razor-cards  

## Consistency checklist (run before accepting art)

- [ ] Correct character ID costume present  
- [ ] Face markers intact (Alice: keyhole brands + scar; Red: freckles/dirt + exhausted glare)  
- [ ] Signature prop present if scene requires it  
- [ ] Style matches frenetic/scratchy ink  
- [ ] No accidental “pretty clean anime” drift  
- [ ] Era = 1888 Overgrown Wonderland (unless Leap says otherwise)  

## Negative prompt block (append often)

```
wrong hair color, clean anime face, soft disney eyes, modern clothing,
pristine costume, smiling cheerful expression, glowing magic wands,
bright pastel wonderland, photorealistic, 3d render, chibi, extra fingers,
missing axe (if Red armed), missing vials (if Alice armed), removed scar,
removed keyhole brands, removed porcelain mask (if Stalker)
```

## Writing style (scripts)

- Short punchy dialogue.  
- Captions can be mythic/poetic — dialogue should sound exhausted and sharp.  
- Alice: clipped, brilliant, slightly unhinged.  
- Red: blunt, protective, tired commander energy.  
- Sound effects: industrial / wet / metallic — not cartoony “BOING.”
